## 🔍 TẠI SAO TEAMID BẮT ĐẦU TỪ 3?

**Nguyên nhân có thể:**

1. **Seed data chạy nhiều lần** → Insert failed → Sequence vẫn tăng
2. **Test data cũ bị xóa** → Rows deleted nhưng sequence không reset
3. **Manual insert trước đó** → Sequence đã increment

**PostgreSQL SERIAL behavior:**
```sql
INSERT INTO team (...) VALUES (...);  -- teamid = 1 (sequence: 1 → 2)
INSERT INTO team (...) VALUES (...);  -- teamid = 2 (sequence: 2 → 3)
DELETE FROM team WHERE teamid IN (1,2);  -- Xóa rows nhưng sequence vẫn là 3
INSERT INTO team (...) VALUES (...);  -- teamid = 3 (sequence: 3 → 4)
```

## 🛠️ GIẢI PHÁP

### Option 1: Reset Sequence (Khuyến nghị cho development)

**Thêm vào Program.cs seed section:**
```csharp
// Reset team sequence để teamid bắt đầu từ 1
await context.Database.ExecuteSqlRawAsync(@"
    DELETE FROM team;  -- Xóa tất cả team cũ
    ALTER SEQUENCE team_teamid_seq RESTART WITH 1;  -- Reset sequence về 1
");

// Sau đó insert team data
await context.Database.ExecuteSqlRawAsync(@"
    INSERT INTO team (areaid, name, teamtype) 
    SELECT a.areaid, v.name, v.teamtype::team_type
    FROM area a, (VALUES
        ('Khu vực 1A', 'Team Alpha', 'Main'),
        ('Khu vực 1A', 'Team Beta', 'Support'),
        ...
    ) AS v(areaname, name, teamtype)
    WHERE a.name = v.areaname;
");
Console.WriteLine("✅ Team data seeded successfully (starting from teamid=1)");
```
